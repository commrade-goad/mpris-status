# MPRIS-STATUS
a simple program that will output the current media metadata or status.

## BUILD

```sh
cargo build --release
```

## USAGE

to display status :

```sh
./build/release/mpris-status s
# or
./build/release/mpris-status status
```


to diplay metadata :

```sh
./build/release/mpris-status m
# or
./build/release/mpris-status metadata
```
